---
title: "Flujo de Trabajo Recomendado"
---

A continuación se describe el flujo estándar para preparar, aprobar y cerrar una cotización de viaje:
<table class="manual-table"><tr><td>

**#**
</td><td>

**Acción**
</td><td>

**Detalle**
</td></tr><tr><td>

**1**
</td><td>

**Verificar / crear el cliente**
</td><td>

Vaya a CRM y compruebe si el cliente existe. Si no, créelo con + Nuevo Cliente.
</td></tr><tr><td>

**2**
</td><td>

**Crear la cotización**
</td><td>

En Cotizaciones &rarr; + Nueva Cotización. Complete los datos generales y haga clic en Guardar.
</td></tr><tr><td>

**3**
</td><td>

**Agregar ítems al itinerario**
</td><td>

Use + Agregar Ítem para incluir servicios, hoteles o paquetes del catálogo. Use filtros $categoria y $ciudad para buscar rápido.
</td></tr><tr><td>

**4**
</td><td>

**Revisar el resumen financiero**
</td><td>

Verifique el Total a Pagar, IVA y comisión antes de continuar. (Solo visible para Admin y Operations)
</td></tr><tr><td>

**5**
</td><td>

**Solicitar revisión**
</td><td>

Si requiere aprobación, use Solicitar Revisión. El aprobador recibirá una notificación en el sistema.
</td></tr><tr><td>

**6**
</td><td>

**Aprobar la cotización**
</td><td>

El usuario con rol Admin (o permiso quotes.approve) cambia el estado a Aprobada.
</td></tr><tr><td>

**7**
</td><td>

**Enviar al cliente**
</td><td>

Use Responder para enviar por correo o genere el PDF para compartirlo manualmente. (Botón disponible para Admin y Operations)
</td></tr><tr><td>

**8**
</td><td>

**Cerrar la venta**
</td><td>

Una vez confirmado el pago por el cliente, use Cerrar Venta para registrar la operación como exitosa.
</td></tr></table>
